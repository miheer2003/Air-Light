## Overview

AirLight is a desktop application that uses a computer's webcam to recognize hand gestures and control a Tuya-based Wi-Fi Smart RGB Bulb in real time. It uses OpenCV and MediaPipe for local gesture recognition and TinyTuya for local network bulb control. The UI is built using CustomTkinter.

## Data Flow

1. `CameraStream` continuously grabs frames from the webcam in a separate thread.
2. The `AirLightApp` main process loop pulls the latest frame and passes it to `HandTracker`.
3. `HandTracker` yields hand landmarks which are analyzed by `GestureClassifier`.
4. `GestureClassifier` identifies the gesture (e.g. Open Palm, Pinch) and potential brightness values.
5. The gesture string and brightness value are sent to `GestureMapper`.
6. `GestureMapper` acts on the gesture (in a separate thread) and triggers the appropriate command in `BulbController`.
7. `BulbController` sends network commands to the smart bulb via TinyTuya.
